**サービス概要**
 <p>ギャンブル依存症診断と依存症専門病院検索</p>

**ユーザーが抱える課題**
<p>厚生労働省の調査によるとギャンブル依存症の人は推計320万人も存在している。現在ほとんどの人は適切な治療を受けておらずギャンブルが原因で犯罪や家庭崩壊などが発生している。</p>

**解決方法**
 <p>ギャンブルをしている本人や家族にギャンブル依存症の人は病気だということを自覚させ病院に行かせること</p>
